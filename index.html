Room monitoring

Three subsystems are involved in this project:

  • S1-RaspberryPi contains the main logic and stores the events received (temperature, presence, alarms) using an MVC model.
  • S2-Arduino has a servo to raise the alarm, proximity sensor to detect the presence, a temperature sensor, and a Bluetooth module to communicate with the other two subsystems.
  • S3-Android app controls the servo motor in S2 in case of an alarm using an asynchronous message protocol.
