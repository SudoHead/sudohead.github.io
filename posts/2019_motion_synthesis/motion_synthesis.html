Character motion synthesis

Using motion capture data to generate new animation


I am developing a motion synthesis technique for creating realistic human animation in OpenGL.
My current approach is based on motion graphs. A motion graph is a directed graph where each edge represent an animation (or clip), and each node is the connecting point to other similar animations.
Transitions are created to blend similar portions of clips using linear interpolation, which can produce good results if the difference between the two motions is small.

The motion capture data is provided by the CMU Graphics Lab, as a free to use mocap library.

The project is still work-in-progress.
