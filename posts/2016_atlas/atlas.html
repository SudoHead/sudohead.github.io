ATLAS | N-Body simulation in Java

N-body simulation of the solar system

I think that simulators are fantastic tools to learn how complex systems work. We all know the Moon orbits around the Earth, and both are orbiting around the Sun. But what happens if the Moon had the same mass as the Earth, and what if we had two Suns?
That's why I decided to make a sandbox physics simulator for solar systems, where one can add, remove, modify celestial bodies like planets, stars, or even black holes. The main source of inspiration is Universe Sandbox 2.

The code is entirely written in Java 8, using the JavaFX graphics library. Extensibility was a particular focus of this project, which follows the MVC (Model-View-Controller) pattern, allowing for extensions in the model without affecting the rendering process. The model makes use of the strategy pattern for different types of simulation algorithms (e.g. brute-force or Barnes-Hut) and collision algorithms, that can be replaced "on the fly" changing the overall behaviour of the system.
